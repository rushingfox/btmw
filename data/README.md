# `data/`

This directory holds two kinds of data files:

- **`data/external/`** — third-party reference curves (digitized from
  published figures or supplied by the original authors). These are
  tracked in git. See [data/external/README.md](external/README.md) for
  per-file citations.
- **`data/cache/`** — checked-in intermediate `.npz` / `.txt` files written by
  the Type-1 `extract()` step of each figure pipeline. These caches let the
  default plotting commands regenerate the paper figures quickly without raw
  simulation outputs. They can be regenerated by running the relevant
  `btmw <command> --refresh` against the raw simulation outputs, or by
  submitting one of the sbatch scripts in [scripts/](../scripts/).

The Type-2 `plot()` step of each figure reads from `data/cache/` and
optionally `data/external/`, and writes to `figures/`.

> **Exception — `projection-map`**: this subcommand has no cache layer
> and no subdirectory here. It reads directly from the SWIFT snapshot
> every time it is run.
